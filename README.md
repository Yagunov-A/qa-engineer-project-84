## Состав репозитория:
Файл __REDME.md__ - этот файл  
Файл __about.yml__ - кратко об исполнителе  
Файл __requiments.yml__ - требования к приложению  
Файл __test-cases.yml__ - тест кейсы  
Файл __testing-report.yml__ - отчёт о тестировании  
Файл __bugreports.yml__ - отчёт о выявленных недостатках  
Файл __regress-report.yml__ - отчёт о проведённом регрессионном тестировании v2 приложения (о тестировании + о выявленных недостатках)  
Каталог __screens of bug__ - скриншоты экрана приложения с выявленными недостатками  
Каталог __screens of bug v2__ - скриншоты экрана приложения v2 с выявленными недостатками
